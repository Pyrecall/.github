# PyRecall

> **Open-source tools for continual learning and modern AI fine-tuning.**

> **Founded by Siddharth Lakshmi Narayanan, Atharv Ranjan, and Shreyas Gandhe**

PyRecall is an open-source organization dedicated to developing tools that improve the fine-tuning of machine learning models. Our work focuses on reducing **catastrophic forgetting**, enabling developers and researchers to adapt models to new tasks while preserving previously learned knowledge.

---

## Mission

Our mission is to make continual learning practical, accessible, and reproducible through high-quality open-source software.

We believe AI development should be:

* Open
* Research-driven
* Reproducible
* Accessible
* Community-focused

---

## Projects

### PyRecall

PyRecall is our flagship Python library for continual learning and modern fine-tuning workflows.

Current capabilities include:

* LoRA and QLoRA fine-tuning
* Built-in evaluation metrics
* Checkpoint management
* Command-line interface
* Modular architecture for research and experimentation
* Comprehensive documentation

As the organization grows, additional libraries and tools will be developed under the PyRecall ecosystem.
